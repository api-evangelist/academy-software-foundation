# Academy Software Foundation (academy-software-foundation)
The Academy Software Foundation (ASWF) is a Linux Foundation project supporting open source software development for the motion picture and visual effects industries. It hosts projects like OpenEXR, OpenVDB, OpenColorIO, and OpenTimelineIO used in film and media production pipelines.
